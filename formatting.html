HTML Formatting

This page shows different ways to format content in HTML without using CSS. I will demonstrate preformatted text, blank spaces, a div element, and background color.

1. Preformatted Text

The pre tag keeps all spacing and line breaks exactly as you type them. It is useful for showing code or any text where spacing matters.

2. Blank Spaces Using &nbsp;

Normal spaces in HTML collapse into one space. To add extra spaces you use the &nbsp; character which stands for non-breaking space.

3. A Div Element

A div is a container that groups content together. On its own it does nothing special but it can hold other elements and be styled later.


Inside a Div

This paragraph and heading are both inside a div element. The div is grouping them together as one block of content.

4. Background Color

The background color of this page is set to lightblue using the bgcolor attribute in the body tag. You can see it is different from the other pages on this site which use lightyellow.


To set a background color in HTML you write:

<body bgcolor="lightblue">
