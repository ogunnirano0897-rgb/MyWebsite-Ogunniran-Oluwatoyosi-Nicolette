3. Definition List

A definition list shows words and what they mean. I used it as an HTML glossary.


HTML
HyperText Markup Language. The language used to build web pages.
Tag
A keyword in angle brackets that tells the browser what to do.
Attribute
Extra information added inside a tag to change how it behaves.
Browser
A program like Chrome or Firefox that reads and displays web pages.
DOCTYPE
A line at the top of every HTML file that tells the browser which version of HTML is being used.
