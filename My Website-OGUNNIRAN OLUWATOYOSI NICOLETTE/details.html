What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. Every website you visit on the internet is built using HTML as its foundation.


HTML uses tags to tell the browser how to display content. Tags are written inside angle brackets like this: <p> for a paragraph or <h1> for a main heading.

How HTML Works

Opening and Closing Tags

Most HTML tags come in pairs. There is an opening tag and a closing tag. The closing tag has a forward slash before the tag name.


For example:

Opening tag: <p>

Closing tag: </p>

The Basic Structure

Every HTML document must have this basic structure:


  1. DOCTYPE declaration - tells the browser it is an HTML5 file
  2. html tag - the root element that wraps everything
  3. head section - contains the title and other invisible information
  4. body section - contains all the content users can see

Why I Find This Topic Interesting

What I like about HTML is that you can see your results immediately in the browser. You write a tag, save the file, refresh the browser and the change appears. That kind of instant feedback makes it easy and exciting to learn.
